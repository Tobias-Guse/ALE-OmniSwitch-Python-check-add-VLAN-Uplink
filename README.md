# ALE-OmniSwitch-Python
This project is created to share and gather experiences for a community interested in Alcatel-Lucent Enterprise network products such as OmniSwitch Release 8, Stellar WLAN, and OmniVista 10, as well as Python, Java scripting, and more.

Feature:
This script checks for existing VLANs on OmniSwitch devices. It runs CLI commands like show vlan and creates a new VLAN if the requested VLAN is not available in the configuration.

Setup: Visual Code, OmniSwitch 8.10R03
